# V3-final Paper Experiment Mapping

## 1. Scope

This mapping explains how MBE artifacts support experiment organization. It does not claim that current outputs are paper-grade final results.

## 2. Runtime Metrics

Runtime summaries support local emulator inspection for throughput, latency, success/failure, node/process, network, shard, committee, and epoch fields.

## 3. Metaverse Workload Organization

V3.13 metaverse artifacts organize synthetic asset/avatar/scene/offchain/cross-metaverse scenarios. They are not real platform traces.

## 4. Baseline Matrix

Baseline matrix and sweep artifacts organize controlled local comparisons. Planned external baselines remain placeholders unless explicitly implemented.

## 5. Fault Observation

V3-final fault artifacts document deterministic node failure/recovery, network delay/drop, target congestion, and Relay fault observations. They are not Byzantine adversary evidence.

## 6. Observability Evidence

Observability artifacts summarize local component health and timelines from generated artifacts. They are not production monitoring.

## 7. Reproducibility Package

The final catalog, manifest, guide, manual, and final summary help rerun and inspect local experiments under the same configuration.

## 8. Limitations

V3-final does not prove production performance, production fault tolerance, production consensus safety, production networking, or real metaverse trace fidelity.
